# Attendance Management API

This is a simple REST API built using Node.js and Express.js for managing user attendance.

It supports basic CRUD operations:
- Create user (POST)
- Get all users (GET)
- Update user (PUT)
- Delete user (DELETE)

The project is designed to understand backend fundamentals such as routing, request & response handling, and HTTP methods.

Note: Data is stored in memory (no database).
